# Fortaleza 1786 — Meniu Digital

Site meniu pentru restaurantul Fortaleza, Oiartzun, Țara Bascilor.

## Limbi
- Español (ES)
- Euskara (EU)
- English (EN)
- Français (FR)

## Cum îl pui pe GitHub Pages

1. Creează un cont pe github.com
2. Click "New repository" → numește-l `fortaleza-1786`
3. Bifează "Add a README file" → Create repository
4. Click "Add file" → "Upload files"
5. Trage `index.html` în fereastră → Commit changes
6. Mergi la Settings → Pages → Branch: main → Save
7. Site-ul va fi live la: https://USERNAME.github.io/fortaleza-1786

## Cum generezi QR-ul

1. Copiază link-ul de mai sus
2. Mergi pe qr-code-generator.com
3. Lipești link-ul → Download PNG sau SVG
